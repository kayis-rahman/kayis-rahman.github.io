I grew up in a small coastal town in Kerala, South India. I write code for a living and have done for ten years.

Right now I'm a Staff Security Software Engineer at Qualcomm in Cork, Ireland, working on PKI, Hardware Security Modules, and secure device provisioning — the infrastructure that establishes cryptographic trust between hardware and backend systems at scale. It's slow, deliberate work with very little margin for error. I like it more than I expected to.

Before Qualcomm I spent a decade moving through the full stack — Java and Spring Boot at TCS and EY, Golang and Android at a video streaming startup in Trivandrum, cloud infrastructure across AWS, GCP and Azure, leading engineering teams of seven, shipping things that mattered to the people using them. I hold an MSc in Information Systems with Computing from Dublin Business School, completed full-time after resigning from my previous role. A decision that felt reckless at the time and turned out to be the right one.

My core technical home is Java and Spring Boot. I've architected microservices platforms, led cloud-native migrations, built full-stack systems with Angular and GraphQL, and automated infrastructure with Docker, Kubernetes, Terraform and CI/CD pipelines. Go is my second language. PKI and cryptographic security is where I'm going deep right now.

On the side I'm building three things. Synapse is a self-hosted AI proxy running on a Raspberry Pi 5 that routes inference intelligently between a local vLLM server and lightweight models — built partly as an infrastructure experiment and partly to stop paying per token. A Pomodoro timer for iOS with an LLM layer that adapts session behaviour based on task type and focus patterns. And an automated swing trading system in Java targeting NSE/BSE equities, using a locally hosted Qwen3 model on dual RTX 5090s as a qualitative sentiment filter — not a signal generator.

I also write about what I'm building. PKI and security infrastructure, self-hosted LLM inference on rented GPUs, Java backend architecture, and occasionally the longer arc of a decade in software. The blog is honest and technical. No content marketing.

Outside work I play and watch football, play cricket, and spend more time than I should on console games. All my devices are Apple — MacBook is where everything gets built. One thing on the longer horizon: building a AAA studio quality game. Not a side project. A proper one.

If something here is relevant to you — whether you're building in a similar space, have a hard problem that needs a security or backend engineering perspective, or just want to talk — reach out.
